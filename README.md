# COVID-19 Symptoms Clustering Analysis 

Этот проект использует машинное обучение (метод K-средних) для группировки данных о симптомах COVID-19. Он позволяет классифицировать пациентов по схожим признакам и визуализировать структуру данных с помощью PCA.

##  Основные возможности
* **Предобработка данных:** Автоматическая очистка и масштабирование признаков с помощью `StandardScaler`.
* **Кластеризация:** Использование алгоритма `K-Means` для разделения данных на 4 группы.
* **Оптимизация:** Определение количества кластеров методом "Локтя" (Elbow Method).
* **Визуализация:** Сжатие данных через PCA (Principal Component Analysis) для отображения 2D-графика кластеров.
* **Экспорт:** Сохранение обученной модели и скалера в формат `.pkl` для дальнейшего использования в продакшене.

## 🛠 Технологии
* **Python 3.x**
* **Pandas / Numpy** — работа с данными.
* **Scikit-learn** — машинное обучение и PCA.
* **Matplotlib** — визуализация графиков.
* **Joblib** — сохранение моделей.
